# Raycast setup

## General setup

+ Toogle with `Cmd + Space`
+ Import rectangle preset for window management (`Settings -> extensions -> window management`)
+ `Cmd + C` : open clipboard
+ `Opt + s` : switch window
+ `Cmd + S` : search screenshots
+ App shortcut
  + `Opt + 1` chrome
  + `Opt + 2` kitty
  + `Opt + 3` obsidian
  + `Opt + 4` slack
